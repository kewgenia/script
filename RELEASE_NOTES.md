# Server Setup - Релиз v2.2.0

## 🛡️ Описание

Модульный инструмент для первоначальной настройки, установки ПО и администрирования серверов под управлением Debian/Ubuntu.

Проект предоставляет интерактивное меню для автоматизации рутинных задач по настройке и администрированию сервера, следуя лучшим практикам информационной безопасности.

## 🚀 Возможности (текущая версия)

### ✅ Реализовано (v2.2.0)

#### 👤 Управление пользователями
*   Создание нового пользователя с настройкой SSH-ключей
*   Управление sudo-правами (добавление/удаление из группы)
*   Генерация SSH-ключей для пользователя

#### 🔒 Настройка SSH и 2FA
*   Смена порта SSH, отключение входа root
*   Настройка SSH-ключей и проверка подключения
*   Двухфакторная аутентификация (Google Authenticator)
*   Настройка PAM и SSH для 2FA

#### 🛡️ Фаервол (UFW)
*   Базовая настройка UFW (правила по умолчанию)
*   Открытие портов: SSH, HTTP, HTTPS, Tailscale
*   Поддержка IPv6 в UFW
*   Добавление пользовательских портов

#### 🛡️ Защита (Fail2Ban/CrowdSec)
*   Установка и настройка Fail2Ban
*   Мониторинг логов UFW, блокировка по IP
*   Альтернатива: CrowdSec (агент + bouncer)
*   Установка коллекций, настройка приема логов

#### 🔄 Автообновления
*   Настройка unattended-upgrades
*   Автоматическая установка обновлений безопасности

#### ⚙️ Система и Ядро
*   Настройка параметров ядра (sysctl) для безопасности
*   Конфигурация hostname, локализации, часового пояса
*   Отключение ненужных системных служб

#### 🌐 Сетевые инструменты
*   Установка Docker (с настройкой daemon.json)
*   Установка Tailscale VPN с автоматической настройкой
*   Альтернатива: NetBird VPN
*   Мониторинг контейнеров (dtop)

#### 💾 Бэкапы (rsync)
*   Настройка автоматических бэкапов через rsync
*   Проверка подключения к удаленному серверу
*   Настройка SSH-ключей для бэкапов

#### ⚙️ Ресурсы (Swap/Время)
*   Создание/настройка swap-файла
*   Синхронизация времени (NTP/timesyncd)

#### 🔍 Аудит и Очистка
*   Lynis аудит безопасности системы
*   Очистка пакетов провайдеров (cloud-init и др.)
*   Финальная очистка системы после настройки

#### 🎨 Кастомизация .bashrc
*   Установка кастомного .bashrc с алиасами
*   Полезные функции для системного администрирования

#### 🔄 Обновление системы
*   Полный цикл: `apt update`, `upgrade`, `autoremove`
*   Очистка кэша, проверка необходимости перезагрузки

#### 🌐 Зеркала APT
*   Быстрая замена репозиториев на Yandex Mirror
*   Ручная настройка зеркал
*   Создание бэкапов и восстановление конфигурации

### 🔜 В планах (заглушки в меню)

*   Создание пользовательского дашборда (dashboard)
*   Плагины для виджетов дашборда
*   Расширение системы конфигурации (`config/server.conf`)

## 📋 Требования

*   **ОС:** Debian 12+ или Ubuntu 20.04+
*   **Права:** Доступ к root (через `sudo`)
*   **Сеть:** Интернет-соединение для установки пакетов

## 🚀 Быстрый старт

### Установка одной командой

```bash
bash <(curl -Ls https://raw.githubusercontent.com/kewgenia/script/main/install.sh)
```

### Установка вручную

```bash
# 1. Скачайте скрипт установки
curl -O https://raw.githubusercontent.com/kewgenia/script/main/install.sh

# 2. Сделайте его исполняемым
chmod +x install.sh

# 3. Запустите от имени root
sudo ./install.sh
```

### Использование

После установки команда `server-setup` будет доступна из любого места:

```bash
sudo server-setup
```
