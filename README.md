# VPS Security Initial Setup

Скрипт для первоначальной настройки безопасности VPS сервера с нуля. Проект использует модульную архитектуру с автоматической генерацией меню через манифесты.

## 🚀 Возможности

### ✅ Реализовано
*   **Обновление системы** — полное обновление пакетов Debian/Ubuntu (`apt update/upgrade/dist-upgrade`, очистка).
*   **Управление зеркалами APT** — замена репозиториев на Yandex Mirror или ввод пользовательских зеркал, создание бэкапов и восстановление.

### 🔜 В планах (заглушки в меню)
*   Создание пользователя с ограниченными правами.
*   Настройка SSH-ключей и безопасности SSH (порт, root login).
*   Настройка фаервола (UFW/Firewalld).
*   Установка и настройка Fail2ban.
*   Отключение ненужных служб.
*   Настройка автоматических обновлений.
*   Дополнительные меры защиты (2FA, права доступа).

## 🚀 Быстрый старт

### 1. Установка
Скрипт устанавливается в `/opt/vps-security-setup` и создает симлинк для быстрого запуска.

```bash
sudo bash install.sh
```

### 2. Запуск
После установки запустите скрипт одной из команд:

```bash
sudo vps-security-setup
# или
sudo /opt/vps-security-setup/vps-security-setup.sh
```

## 🛠️ Разработка

Проект следует стандартам `STYLE_GUIDE.md`:
*   Используется `#!/bin/bash`.
*   Вывод через `info`, `ok`, `warn`, `err`.
*   Ввод через `safe_read`, `ask_yes_no`.
*   Модули подключаются через `source` и используют манифесты `@menu.manifest`.

### Добавление нового модуля
1. Создайте файл в `modules/security/`.
2. Добавьте блок `@menu.manifest`.
3. Реализуйте функции.
4. Подключите модуль в `vps-security-setup.sh`.
